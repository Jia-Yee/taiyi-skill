# 太乙 (Taiyi) — 虚拟历史人物人格技能

> 召唤历史上的伟大思想者，与他们对话，向他们请教。

---

## 简介

**Taiyi（太乙）** 是一个运行在 [OpenClaw](https://github.com/openclaw) 上的 AI 技能（Skill）。它能够通过网络搜索或用户上传的文档，构建历史人物的知识体系，学习其思考方式，形成独立的虚拟人格——让你可以真正地与欧几里得探讨几何，与苏格拉底辩论哲学，与牛顿推演物理。
示例以QClaw测试。

---

## ✨ 功能特性

### 🔍 人物信息收集
- **网络搜索**：输入人物名字，自动搜索其生平、著作、思想
- **文档上传**：上传 PDF、TXT、DOCX 等格式的著作或资料，AI 自动提取关键信息

### 🧠 知识结构化
将收集到的信息整理为结构化知识库：
- 生平与年代背景
- 代表著作与核心观点
- 思想体系与思维方式
- 对话人格定义（说话风格、性格特征、知识边界）

### 🎭 人格塑造
- 根据历史记载塑造人物性格与说话风格
- 严格基于资料，不虚构信息
- 记录所有信息来源，便于验证

### 📐 数学知识图谱（数学家专属）
对于数学相关人物，Taiyi 会额外：
- 将人物与现有数学知识图谱建立关联
- 识别并标注重要定理、概念、证明
- 模拟人物的数学逻辑思维
- 支持推导新问题，让"欧几里得"真正思考数学

## 🚀 快速开始

### 安装

将本技能安装到你的 OpenClaw 环境：

```bash
# 将 taiyi 目录放入 skills 目录
cp -r taiyi ~/.agents/skills/taiyi
```

或直接安装打包好的 `.skill` 文件（如有）：

```
在 OpenClaw 控制台中：安装技能 → 选择 taiyi.skill
```

### 使用示例

**创建人物人格：**
```
用户：创建欧几里得人格
用户：我想和牛顿对话
用户：帮我学习柏拉图的思想
```

**上传文档创建：**
```
用户：[上传《几何原本》PDF] 基于这本书创建欧几里得人格
```

**与人物对话：**
```
用户：欧几里得，什么是几何学的第一性原理？
欧几里得：一切几何学皆从公理出发。我在《几何原本》中列出了五条公设……
```

**请教数学问题：**
```
用户：欧几里得，你能证明素数有无穷多个吗？
欧几里得：当然。假设素数有限，设它们为 p₁, p₂, …, pₙ……
```

## 📁 数据存储结构

人物知识库存储在本地：

```
~/.qclaw/taiyi/
└── 欧几里得/
    ├── info.json          # 基础信息与人格定义
    ├── works/             # 著作文本
    ├── knowledge/         # 结构化知识
    └── persona.md         # 人格描述
```

---

## 🛠️ 脚本工具

技能内置 `scripts/taiyi.py`，可通过命令行管理人物人格：

```bash
# 列出所有已创建的人物
python scripts/taiyi.py list

# 创建人物
python scripts/taiyi.py create 欧几里得

# 查看人物信息
python scripts/taiyi.py get 欧几里得

# 删除人物
python scripts/taiyi.py delete 欧几里得
```

## ⚠️ 注意事项

1. **知识边界**：Taiyi 严格基于收集到的资料回答，不会凭空虚构历史事实
2. **资料溯源**：所有信息均记录来源，可追溯验证
3. **数学严谨**：数学问题的回答保证逻辑正确性
4. **持续丰富**：可以通过上传更多资料来丰富人物的知识库
