# 阿基米德数学知识图谱

## 人物节点

```
阿基米德 (Archimedes)
├── 活跃时期：公元前287年 - 公元前212年
├── 出生地：叙拉古（今意大利西西里岛）
├── 主要领域：几何学、物理学、工程学
├── 称号："力学之父"
└── 与牛顿、高斯并列世界三大数学家
```

## 学术传承关系

```
欧几里得 ──→ 埃拉托塞（欧几里得学生）──→ 阿基米德
    │                                       │
    │                                       │
    └──────────────────────────────────────────┘
         （阿基米德继承几何学，并加入物理维度）
```

| 阿基米德发现 | 现代数学对应 | 影响程度 |
|--------------|-------------|---------|
| 穷竭法 | 积分学（牛顿-莱布尼茨） | 微积分先驱 |
| 圆周率逼近 | π 的数值计算 | 至今使用22/7 |
| 球体积公式 | V = (4/3)πr³ | 几何经典 |
| 浮力定律 | 流体静力学 | 船舶理论基础 |
| 杠杆原理 | 静力学平衡 | 机械基础 |
| 螺旋线 | 阿基米德螺旋曲线 | 数学曲线研究 |

## 数学贡献图谱

### 几何学领域
```
平面几何
    ├── 圆面积 → 多边形逼近法 → π 计算
    │
立体几何
    ├── 球体积 → 外接圆柱 → 2:3比例
    ├── 圆柱体积
    └── 旋转体体积 → 穷竭法 → 积分雏形
```

### 数论领域
```
圆周率
    ├── 内接多边形 → 下界 π > 223/71
    ├── 外切多边形 → 上界 π < 22/7
    └── 精度：小数点后2位准确
```

### 物理学领域
```
静力学
    ├── 杠杆原理 → 力矩平衡
    ├── 重心计算 → 几何图形重心
    └── 滑轮系统 → 机械设计
    │
流体力学
    ├── 浮力定律 → 排水重量 = 浮力
    ├── 液体比重测定
    └── 物体漂浮稳定性判据
```
