# Taiyi - 虚拟人物人格创建与管理

## 概述

Taiyi 能够通过以下方式创建虚拟人物人格：
1. **网络搜索** - 输入人物名字，自动搜索其生平、著作、思想
2. **文档上传** - 上传人物的著作、笔记、信件等文档
3. **知识整合** - 将所有信息整合为结构化知识体系
4. **人格形成** - 学习人物的思考方式，形成独特的对话人格

## 核心能力

### 1. 人物信息收集

**触发方式：**
- 用户输入人物名字（如"欧几里得"、"牛顿"、"柏拉图"）
- 用户上传文档（PDF、TXT、DOCX等）

**工作流程：**
1. 识别人物身份和领域
2. 使用网络搜索收集人物基本信息、生平、著作
3. 处理用户上传的文档，提取关键信息
4. 整理所有资料来源

### 2. 知识结构化

将收集到的信息整理为结构化格式：

```
人物知识库/
├── 基础信息/
│   ├── 生平.md
│   ├── 年代.md
│   └── 主要成就.md
├── 著作/
│   ├── 著作列表.md
│   └── 核心观点.md
├── 思想体系/
│   ├── 核心思想.md
│   ├── 思维方式.md
│   └── 代表性观点.md
└── 对话人格/
    ├── 性格特征.md
    ├── 说话风格.md
    └── 知识边界.md
```

### 3. 人格塑造

根据收集到的信息塑造独特的人格：

- **说话风格**：根据人物身份调整（学者式、对话式、讲授式）
- **知识边界**：明确人物的知识范围，避免虚构
- **性格特征**：根据历史记载塑造人物性格
- **思维模式**：学习人物的思考方式，用于回答问题

### 4. 数学人物特殊处理

对于数学相关人物：

1. **知识图谱关联** - 将人物与现有数学知识图谱建立关系
2. **数学贡献识别** - 识别并标注重要定理、概念、证明
3. **逻辑思维模拟** - 学习人物的数学思维模式
4. **问题回答** - 能够回答相关数学问题，甚至推导新问题

## 使用流程

### 第一步：创建人物

用户输入：
- "创建欧几里得人格"
- "我想和柏拉图对话"

系统将：
1. 搜索人物基本信息
2. 查找其代表著作
3. 收集思想观点

### 第二步：知识构建

系统自动：
1. 创建人物专属文件夹
2. 整理所有收集到的信息
3. 标注关键概念和思想

### 第三步：人格激活

完成后，用户可以：
- 直接与人物对话
- 请教专业问题
- 探讨思想观点

## 对话示例

**用户**：创建欧几里得人格

**系统**（完成创建后）：
```

📚 知识库已建立：
- 生平：《几何原本》作者，古希腊数学家
- 思想：公理化体系、逻辑推理
- 风格：严谨、讲授式、喜欢用几何证明

您现在可以与欧几里得对话了！
试试问："什么是几何学的第一性原理？"
```

## 数据存储

人物知识库存储在：
```
~/.qclaw/taiyi/人物名/
├── info.json          # 基础信息
├── works/             # 著作文本
├── knowledge/         # 知识结构
└── persona.md         # 人格定义
```

## 注意事项

1. **知识边界**：严格基于收集到的资料，不虚构信息
2. **资料溯源**：记录信息来源，便于验证
3. **持续学习**：可以通过更多对话丰富人格
4. **数学严谨**：数学问题回答需保证准确性
